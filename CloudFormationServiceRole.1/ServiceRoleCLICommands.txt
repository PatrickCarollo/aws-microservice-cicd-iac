CLI COMMANDS FOR INITIAL CLOUDFORMATION STACKS' IAM Roles




Create roles:

1.
aws iam create-role --role-name MainCICDStackServiceRole --assume-role-policy-document '{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": {
                "Service": "cloudformation.amazonaws.com"
            },
            "Action": "sts:AssumeRole"
        }
    ]
}'
